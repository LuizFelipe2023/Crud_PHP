# Simple CRUD Application

A basic CRUD (Create, Read, Update, Delete) application in PHP with MySQL.

## Features

- CRUD operations: Create, Read, Update, Delete user records.
- Minimalistic UI with HTML and CSS.
- Uses PHP and MySQL.
